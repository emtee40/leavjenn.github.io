M3U8 Downloader Plugin List

Usage:

1. Install the plugins from Google Play.

2. Open m3u8 Downloader, click "Choose" plugin button at top right, select the plugin from the pop-up list dialog.

3. Paste the link that contains the video from the website you selected to the "URL" input field, click "Extract video".

4. Wait for the extracting. If it works, you'll get a list of all resolution versions of the video. Click one to download.
